Full Design Summary

5 changes across the app — review and approve to proceed

1. Fix Flash on Load

Problem: Splash uses layout="centered", then login/landing switch to "wide". This config switch causes a visible layout jump. The cookie auto-login adds extra rerun cycles.

Fix:

  • Use layout="wide" everywhere — splash, login, landing, tools. No more config switching.
  • The login page will use CSS to center its split-screen layout within the wide canvas (easy since it's all custom HTML anyway).
  • Reduce splash time.sleep(3) to time.sleep(2) — 3s feels too long.
  • Add a CSS class to hide content during rerun transitions (opacity 0 → 1 fade) so the "glimpse" is invisible.

2. Login Page — Immersive Split (Option B)

  • Left panel (60%): Immersive branding — logo + "Stone Harp Analytics" top-left, large hero text "Intelligence Reimagined" centered, concentric decorative rings, stat counters (3 Tools / 24/7 / Secure) at bottom, subtle grid overlay
  • Right panel (40%): "Welcome back" heading, username + password inputs, remember-me checkbox, Sign In button, footer text
  • Implementation: Full-width HTML/CSS in _login_page() using st.markdown(unsafe_allow_html=True) for the left panel, then Streamlit form widgets for the right panel via st.columns([1.2, 0.8])
  • Hide sidebar + header via CSS (already done)
  • Entry animations: fade-up for left content, slide-in for right form

3. Page Transitions

  • Entry animation: Keep and refine pageIn CSS (fade + slide up + blur). Increase duration slightly (0.5s → 0.6s) for smoother feel.
  • Exit dimming: The existing CSS .stApp[data-test-script-state="running"] dims to 0.6 opacity — change to 0.85 so it's subtler, less jarring.
  • Gold shimmer bar: Already exists during reruns. Keep as-is — it's the best transition indicator Streamlit allows.
  • Remove JS page-loader: The components.html() script in _load_css() fights Streamlit's own mechanism and adds a hidden iframe. Remove it — the CSS-only approach is cleaner and more reliable.

4. Landing Page — Equal 3-Card Row (Option A)

  • 3 equal cards in st.columns([2, 2, 2]) with small gutters
  • Topbar integration: Move Sign Out into the topbar (next to user badge) — eliminate the awkward standalone Sign Out button below cards
  • Admin button: Move into topbar as a subtle gear icon (admin only), replacing the separate button row
  • Staggered card animations: 0.25s → 0.35s → 0.45s delay
  • Daedalus card: amber accent (#f59e0b), grid icon — already done

5. Sidebar Polish

  • Active page highlight: Add a left gold border + slightly brighter background on the active nav link
  • Hover states: Smoother transitions on nav links (0.15s → 0.25s), subtle background shift
  • User info section: Restyle the bottom sidebar info — add the gold avatar circle, role badge, cleaner layout matching the topbar style
  • Back to Home: Style as a distinct "← Back" link instead of a regular nav item
